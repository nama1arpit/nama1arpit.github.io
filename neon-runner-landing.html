Neon-Runner GitHub

A neon themed 2d platformer game made in pygame

Gameplay

Currently, there is just a single boss fight. There are two pickup items:

1. Coin: Adds up to the player score
2. Lightning: Allows the player to activate lightning fast speed for a few seconds

Attack

Currently, the player can only attack the enemy by jumping on them

Lightning Speed

The little lightning collectable allows to run at lightning speed and gives the player an edge over the boss.

Controls

1. Left and Right Arrows - Movement
2. Space - Jump
3. Shift - Activate lightning fast speed

Directions to Run (Linux)

1. Download and extract the zip file here
2. Run the command chmod u+x neon-runner. You need to run this only once as it allows the file to be run as an executable.
3. Run the command ./neon-runner
